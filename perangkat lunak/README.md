# WasteSmart Pro

## Deskripsi
WasteSmart adalah aplikasi pengelolaan sampah berbasis komunitas yang membantu masyarakat mengelola sampah dan mendapatkan poin reward.

## Fitur
1. Register & Login user
2. Input data sampah
3. Sistem poin reward
4. Tracking data sampah
5. Dashboard pengguna
6. Penyimpanan database MongoDB
7. API backend
8. Notifikasi sederhana
9. Riwayat data
10. UI sederhana
11. Sistem user
12. Penyimpanan data real-time
13. Struktur MVC backend
14. Pengolahan data sampah
15. Integrasi frontend-backend

## Teknologi
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Cara Menjalankan

### Backend
cd backend
npm install
node server.js

### Frontend
Buka file index.html di browser
